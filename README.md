# Introduction

This is a simple example application to read the (partial, randomized) twitter data stream for a given list of keywords.

# Configuration

Insert your developer- and application-specific values for

    consumerKey=...
    consumerSecret=...
    token=...
    secret=...

and add your keywords to

    keywords=
